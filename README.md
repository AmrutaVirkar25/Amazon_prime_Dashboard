# Amazon_prime_Dashboard
This Power BI Dashboard provides an analytical overview of Amazon Prime Video user behavior, subscription trends, and churn rate. It enables stakeholders to monitor key business metrics and understand customer engagement patterns across different demographics and subscription plans.
